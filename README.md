# Halftone Lab

Image-based SVG halftone generator. Upload an image, pick a pattern (dots, squares, triangles), tweak tone and grid, and export SVG or PNG. Fully client-side.

## Stack

Vite · React · TypeScript · Tailwind · shadcn/ui · Zustand

## Development

Requires Node.js and [pnpm](https://pnpm.io/).

```sh
pnpm install
pnpm dev        # dev server on http://localhost:8080
pnpm build      # production build
pnpm lint       # eslint
pnpm test       # vitest single run
pnpm test:watch # vitest watch
```
